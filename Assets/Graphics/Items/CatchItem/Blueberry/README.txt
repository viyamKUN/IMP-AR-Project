**Gourmet Blueberry.**

*Single California Grown Gourmet Blueberry.*

*2 x 2 x 1.5cm (70 micrometers per texel @ 512x512)*

Scanned using physically based process developed by inciprocal Inc. that enables highly photo-realistic reproduction of real-world products in virtual environments. Our hardware and software technology combines advanced photometry, structured light, photogrammtery and light fields to capture and generate accurate material representations from tens of thousands of images targeting real-time and offline path-traced PBR compatible renderers.

Zip file includes low-poly OBJ mesh (in meters) with a set of 512x512 PBR textures compressed with *lossless* JPEG (no chroma sub-sampling).
